# Import Manifests

Import manifests document maintainer-run source imports.

Use `source-import.template.json` as a starting point and save real manifests with a stable name:

```text
{source-id}-{yyyy-mm-dd}.json
```

Examples:

```text
ministry-universities-2026-06-28.json
wikidata-universities-2026-06-28.json
```

Do not create manifests for sources that are not approved or under active maintainer review.
