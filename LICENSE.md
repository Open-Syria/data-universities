# License

OpenSyria Data Universities uses separate licensing for data and code.

## Data and Documentation

Released data artifacts and documentation are intended to be published under:

```text
Creative Commons Attribution 4.0 International (CC BY 4.0)
```

License text:

```text
https://creativecommons.org/licenses/by/4.0/
```

Attribution to OpenSyria and source datasets must be preserved.

## Scripts and Configuration

Repository scripts and configuration files are licensed under:

```text
MIT
```

## Source-Specific Requirements

Some upstream sources require attribution or additional handling.

OpenStreetMap-derived data is licensed under ODbL and must not be included in the default CC BY release unless the maintainer explicitly approves the licensing approach for that release.

Each release manifest must list the sources used by that release.
